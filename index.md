## About Me

I am a Structural Analysis Engineer at X-Energy with expertise in finite element analysis (FEA), computational modeling, and machine learning for engineering simulations.

I completed my Ph.D. in Engineering Science and Mechanics at Penn State, where my research focused on integrating machine learning with physics-based simulations to accelerate microstructure modeling and predictive analysis in materials science.

My work combines traditional finite element methods with modern ML techniques to improve simulation efficiency and enable faster design exploration.

Previously I worked as a Finite Element Analysis Intern at ASML, where I developed automated parametric simulation pipelines integrating CAD, Abaqus, and optiSLang for large-scale design studies.

Before that, I worked with the material modeling team at Ansys developing Python-based workflows for multiscale material calibration using PyAnsys.

My interests include:

- Machine learning for scientific computing
- Finite element modeling and simulation
- Computational materials science
- AI-accelerated physics simulations

## Selected Projects

### Machine Learning Accelerated Microstructure Simulation

- Developed CNN and autoencoder models to predict displacement fields from FEA simulations.
- Reduced data dimensionality by **94% using convolutional autoencoders**.
- Achieved accurate prediction of composite material mechanical response.

### Python-Based Microstructure FEA Framework

- Developed Python framework to generate two-phase microstructures with random particle distributions.
- Integrated simulations using the **MOOSE finite element framework**.
- Generated datasets used for training machine learning models.

### Automated Parametric FEA Pipeline (ASML)

- Built automated Abaqus workflow with **40 parametric CAD design variables**.
- Integrated **Ansys optiSLang** for DOE sampling and optimization.
- Enabled automated CAD → meshing → simulation → performance evaluation.
